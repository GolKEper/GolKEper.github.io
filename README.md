# Как опубликовать портфолио на GitHub Pages

Внутри архива:

```
index.html        — сама страница
videos/           — три твоих ролика
posters/          — кадры-обложки (показываются до запуска видео)
README.md         — эта инструкция
```

## Шаг 1. Замени 3 места в index.html

Открой `index.html` в любом редакторе (или потом прямо на GitHub — иконка карандаша) и найди пометки `✏️ ЗАМЕНИ`:

| Что найти | На что заменить |
|---|---|
| `https://kwork.ru/user/ТВОЙ_ЛОГИН` | ссылка на твой профиль Kwork |
| `https://t.me/ТВОЙ_НИК` | твой ник в Telegram |
| `ИМЯ ФАМИЛИЯ` | твоё имя |

## Шаг 2. Создай репозиторий

1. Зарегистрируйся / войди на github.com.
2. Нажми **New repository**.
3. Название репозитория — строго `ТВОЙ_ЛОГИН.github.io` (например, логин `ivan-ai` → репозиторий `ivan-ai.github.io`). Именно такое имя даёт короткий адрес сайта.
4. Видимость — **Public** → **Create repository**.

## Шаг 3. Загрузи файлы

1. В репозитории: **Add file → Upload files**.
2. Перетащи в окно загрузки **всё содержимое архива**: `index.html` и папки `videos`, `posters` (ролики по 10–12 МБ — в лимит загрузки через браузер проходят).
3. Нажми **Commit changes**.

## Шаг 4. Проверь сайт

Через 1–3 минуты открой `https://ТВОЙ_ЛОГИН.github.io` — страница должна работать.

Если не открылась: в репозитории **Settings → Pages** → в разделе Build and deployment выбери **Deploy from a branch**, ветка `main`, папка `/ (root)` → Save и подожди пару минут.

## Что дальше

- **Картинки из папки `posters` — это готовые обложки для портфолио Kwork**: добавляешь работу, ставишь jpg обложкой и прикрепляешь к ней соответствующий mp4.
- Ссылку `https://ТВОЙ_ЛОГИН.github.io` вставляй в отклики на бирже Kwork и в описание профиля — заказчик одним кликом видит и видео, и услуги.
- Когда появится живой Telegram-бот или сайт-демо — в `index.html` есть закомментированный блок «ШАБЛОН НА БУДУЩЕЕ»: раскомментируй и подставь ссылки.
- Любые правки потом: открой файл на GitHub → иконка карандаша → исправил → **Commit changes**. Сайт обновится сам через минуту-две.
